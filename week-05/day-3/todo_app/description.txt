       CLI Todo application
    ==========================
     Command line arguments:
      <press any listed key>
    ==========================
    |-l|  Lists all the tasks
    |-a|  Adds a new task
    |-r|  Removes a task
    |-c|  Completes a task
    |-u|  Uncompletes a task
    |-e|  Empty list
    ==========================
